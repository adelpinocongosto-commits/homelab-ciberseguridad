# homelab-ciberseguridad

Laboratorio de ciberseguridad montado en local con VirtualBox para practicar  técnicas de pentesting en un entorno controlado y legal.

## Entorno

- Kali Linux
- Metasploitable 2
- Red interna VirtualBox

## Objetivos

- Enumeración de servicios
- Captura de tráfico con Wireshark
- Análisis de protocolos inseguros
- SMB Enumeration
- Explotación controlada en entorno de laboratorio

## Herramientas utilizadas

- Nmap
- Wireshark
- SMBClient
- OpenSSH
- Metasploit Framework
